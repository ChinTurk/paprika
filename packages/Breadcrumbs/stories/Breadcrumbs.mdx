### Basic Breadcrumbs

```js
<Breadcrumbs>
  <Breadcrumbs.Link href={URL}>Breadcrumb 1</Breadcrumbs.Link>
  <Breadcrumbs.Link href={URL}>Breadcrumb 2</Breadcrumbs.Link>
  <Breadcrumbs.Link href={URL}>Breadcrumb 3</Breadcrumbs.Link>
</Breadcrumbs>
```

### Breadcrumbs will add a tooltip if the text in your Breadcrumb.Link longer than 40 characters

```js
<Breadcrumbs>
  <Breadcrumbs.Link href={URL}>
    Breadcrumb item with long text will have a tooltip. Hover this item to see.
  </Breadcrumbs.Link>
  <Breadcrumbs.Link href={URL}>Breadcrumb 2</Breadcrumbs.Link>
  <Breadcrumbs.Link href={URL}>Breadcrumb 3</Breadcrumbs.Link>
</Breadcrumbs>
```

#### Result

<Preview>

### Render Breadcrumbs.Link using your own components

```js
const MyButton = (children) => <button>{children}</button>

<Breadcrumbs>
  <Breadcrumbs.Link as="button" href={URL}>
    Breadcrumb link using button
  </Breadcrumbs.Link>
  <Breadcrumbs.Link as="a" href={URL}>
    Breadcrumb link using anchor link
  </Breadcrumbs.Link>
  <Breadcrumbs.Link as={MyButton} href={URL}>Breadcrumb 2 using MyButton</Breadcrumbs.Link>
</Breadcrumbs>
```

#### Result

    <Breadcrumbs.Link as="a">Breadcrumb link using anchor link</Breadcrumbs.Link>

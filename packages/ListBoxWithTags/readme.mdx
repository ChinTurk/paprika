# ListBoxWithTags

## Intro

This component compose the ListBox component which means that can be use in a
similar way. Both of them expose the Option and Group component from `<Listbox />`
so if you already known how to use the ListBox you know as well how to use this one.

## Api

The Api for the `<ListBoxWithTags />` is exactly the same one as the `<Listbox />` one, except for
two more properties which are `hasTagInput` and `placeholderForTagInput`, you can read more about it
on the PropsTable description.

Also this component will forward automatically extra properties as `zIndex` and `getScrollContainer` which
are handy for the `<Popover />` component.

A basic use with code:

```js
<ListBoxWithTags>
  <ListBoxWithTags.Option>Option 1</ListBoxWithTags.Option>
  <ListBoxWithTags.Option>Option 2</ListBoxWithTags.Option>
  <ListBoxWithTags.Option>Option 3</ListBoxWithTags.Option>
</ListBoxWithTags/>
```

## Accessibility

_Note_ keyboard interaction on progress

The `<ListBoxWithTags />` inherent all the Accessibility goodies from the `<ListBox />` and adds
the ability to delete selected tags using the `[delete]` key. Let the user navigates between selected
tags as well using `[‚Üê]` and `[‚Üí]` keys.
And if the tag input is turn on, will keep the focus on the input letting the user search or add a new options in cases don't exists under the current selections.

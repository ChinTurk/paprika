# Button

<Props of={Button} />

## Installation

```
yarn add @/paprika/button
```

or

```
npm install @paprika/button
```

## Usage

The `<Button />` component allows users to perform an action on the page you’re viewing.

The Button component has several unique sub-components that can be used depending on the users needs. The alternate Button's that can be used include the following:

- The Basic Button: `<Button />`
- The Close Button: `<Button.Close />`
- The Link Button: `<Button.Link />`
- The Icon Button: `<Button.Icon />`

To add any of the aforementioned buttons to your project simply import the button as follows:

```js
import Button from "@paprika/button";
```

### Basic Button

To add a basic button import the button and add the `<Button>` component to your project.

```js
<Button>Click Me!</Button>
```

#### Result

<Button>Click Me!</Button>

#### Kinds

Differant projects require differant buttons, luckily the Button component can make us of the `kind` property to create a variety of buttons for differant use cases.

```js
<Button kind="Button.types.kind.DEFAULT">default</Button>
<Button kind="Button.types.kind.PRIMARY">primary</Button>
<Button kind="Button.types.kind.SECONDARY">secondary</Button>
<Button kind="Button.types.kind.DESTRUCTIVE">destructive</Button>
<Button kind="Button.types.kind.FLAT">flat</Button>
<Button kind="Button.types.kind.MINOR">minor</Button>
<Button kind="Button.types.kind.LINK">link</Button>
```

#### Result

<Button kind="default">default</Button>
<Button kind="primary">primary</Button>
<Button kind="secondary">secondary</Button>
<Button kind="destructive">destructive</Button>
<Button kind="flat">flat</Button>
<Button kind="minor">minor</Button>
<Button kind="link">link</Button>

#### Sizes

Buttons can be adjusteds to be a variety of sizes using the `size` proeprty or the `isFullWidth` proeprty.

```js
<Button size="Button.types.size.SMALL">small</Button>
<Button size="Button.types.size.MEDIUM">medium</Button>
<Button size="Button.types.size.LARGE">large</Button>
<Button isFullWidth>full width</Button>
```

#### Result

<Button size="small">small</Button>
<Button size="medium">medium</Button>
<Button size="large">large</Button>
<Button isFullWidth>full width</Button>

#### States

Buttons can be display differant states by adjusting making use of the `isActive`, `isPending`, and `isDisabled` properties.

```js
    <Button isActive>active</Button>
    <Button isDisabled>disabled</Button>
    <Button isPending>pending</Button>
```

#### Result

<Button isActive>active</Button>
<Button isDisabled>disabled</Button>
<Button isPending>pending</Button>

#### Submit

To turn a Button into a submit Button by adding the `isSubmit` property.

```js
<form
  onSubmit={event => {
    alert("submitted");
    event.preventDefault();
  }}
>
  <input type="text" defaultValue="press enter while focus" />
  <Button isSubmit isSemantic={false}>
    submit
  </Button>
</form>
```

#### Result

<ButtonSubmit />

### Close Button

<Props of={Button.Close} />

To make use of a Close Button use the `<Button.Close>` element.

```js
<p>
<Button.Close onClick={clickHandler} ref={buttonRef} />
</p>
<p>
<Small>This CloseButton will capture the focus after 1 second.</Small>
</p>
```
